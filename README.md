# social-network-API [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
## Description
    This is a simple backend application utilizing MongoDB and Mongoose ODM to replicate how a social media site might function on the backend. With this users can share their thoughts, react to their friendsâ€™ thoughts, and create a friend list.
